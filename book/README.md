# Офіційна рецензія-анотація

**Назва:** ERP/1: Архітектура, верифікація та безпека державних інформаційних систем

## Загальна характеристика
Підручник являє собою унікальну та всеосяжну працю, присвячену фундаментальним аспектам розробки, розгортання та підтримки захищених інформаційно-комунікаційних систем (ІКС) державного рівня. Робота глибоко розкриває архітектурні, правові та технологічні засади побудови цифрового суверенітету України, спираючись на реальний досвід експлуатації платформ на базі Erlang/OTP.

## Ключові переваги
1. **Комплексність та міждисциплінарність:** Підручник органічно поєднує нормативно-правову базу (закони України, вимоги КСЗІ, ДСТУ, міжнародні стандарти ISO/IEC, NIST) з глибоким технічним описом.
2. **Фокус на формальній верифікації та безпеці:** Особлива увага приділяється математично доведеній коректності систем (формальна верифікація) та криптографічному захисту інформації (Maker/Checker, X.509, ASN.1 DER), що є критичним для реєстрів та судових систем (ЄСІКС).
3. **Практична цінність:** Книга містить конкретні таксономії (наприклад, 10 розділів ЄСІКС, OID профілі безпеки міністерств та судів), описує архітектуру інтеграції («Трембіта») та розгортання реальних систем.
4. **Інституційна глибина:** Детально розібрана модель Національної програми інформатизації (НПІ) та протокол інституційної трансформації (ДІТ), що робить книгу цінною для управлінців рівня CDTO.

## Цільова аудиторія
Підручник рекомендовано для Software та Enterprise архітекторів, фахівців з кібербезпеки, керівників IT-департаментів органів виконавчої та судової влади (ЦОВВ, ОВВ), а також студентів магістратури за спеціальностями «Комп'ютерні науки», «Інженерія програмного забезпечення» та «Кібербезпека».

## Висновок
Дана праця є еталонним посібником з проєктування високонавантажених і критично важливих державних систем. Вона формує системне бачення того, як технології (Erlang, функціональне програмування) здатні забезпечити безперебійність, прозорість та надійний аудит державного апарату.
