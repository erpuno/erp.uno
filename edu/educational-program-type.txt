Довідник «Типи освітніх програм» (EducationalProgramType)
ERP/1: Освіта
1. Загальна інформація

Назва довідника: Типи освітніх програм
Код: EDUCATIONAL_PROGRAM_TYPE
Тип: Реєстрова сутність (Dictionary)
Призначення: Класифікація типів освітніх програм відповідно до Закону України «Про освіту», стандартів спеціальностей та рівнів освіти.
Використання: Навчальні плани, акредитація, контингент, звітність для МОН, ЄДЕБО.

4. Бізнес-правила

Кожна освітня програма повинна мати один основний тип.
Валідація: ects_min повинен відповідати рівню освіти.
При зміні типу — запуск BPMN-процесу перегляду навчальних планів.
Типи використовуються для автоматичної генерації форм звітності.
