Структура довідника «Освітні програми» (EducationalProgram)
ERP/1: Освіта
1. Загальна інформація

Назва довідника: Освітні програми (EducationalProgram)
Тип: Реєстрова сутність (Dictionary)
Призначення: Каталог акредитованих/використовуваних освітніх програм відповідно до Закону України «Про освіту», стандартів вищої/загальної середньої освіти та ліцензійних умов.
Використання: Навчальні плани, контингент, розклад, звітність для МОН, ЄДЕБО, акредитація.

4. Бізнес-правила

Одна освітня програма може мати кілька форм навчання.
При створенні навчального плану обов’язково вказується посилання на освітню програму.
Зміна статусу акредитації → автоматичний запуск BPMN-процесу оповіщення та коригування планів.
Валідація: credits_ects повинен відповідати рівню освіти.
Історія змін фіксується для аудиту та звітності.

5. Erlang Record (приклад)
erlang-record(educational_program, {
    id,
    code,
    name_ua,
    name_en,
    level,
    specialty_code,
    specialty_name,
    qualification,
    duration_years,
    credits_ects,
    form_of_study_id,      % FK to FormOfStudy
    language,
    accreditation_status,
    accreditation_until,
    license_number,
    department_id,
    is_active = true,
    metadata = #{}
}).
