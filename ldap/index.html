SYNRC LDAP — це високопродуктивний IETF 2849 3296 3671-3673 3866 4510-4518 4522 4525 4526 4929 5480 сумісний сервер каталогів з підтримкою SQLITE бекенду у 300 рядках коду на Elixir.

Опис

SYNRC LDAP розроблено для екстремальної продуктивності та простоти. Маючи лише 300 рядків коду на Elixir, він забезпечує повнофункціональний сервер LDAPv3, здатний обслуговувати масштабні каталоги з різними сховищами даних. Сервер використовує Erlang/OTP для паралелізму та надійності, що робить його стійким для критично важливих застосунків.

Ключові можливості

Бекенди

  • SQLITE (Exqlite)
  • LMDB, MONGODB
  • ROCKSDB NVME

Відповідність

  • IETF LDAPv3
  • RFC 4510—4518
  • RFC 5480, 4929

Протокол

  • ASN.1 (Сертифіковано)
  • Simple Bind, Search
  • Add, Delete, Modify

Продуктивність

  • 300 рядків Elixir
  • Оптимізовано для NVME
  • Висока швидкість

Сервіси та протоколи

  • LDAPv3 — підтримка протоколу для доступу до каталогів (RFC 4511).
  • LDIF — стандартний формат обміну даними для LDAP (RFC 2849).
  • CONF — гнучке керування конфігурацією для екземплярів сервера.
  • SCHEMA — надійна підтримка стандартних та користувацьких схем LDAP.
  • ARCH — масштабована архітектура зберігання для розподілених середовищ.

Історія розробки

  • 2010—2018: Оригінальна реалізація LDAP на Erlang.
  • 2023: LDAP Compliance, реліз 13.7.24.
  • 2024: реліз 15.1.1, інтеграція ASN1.EX X.680.
  • 2026: реєстрація авторського твору.
